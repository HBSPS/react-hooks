# useInput

비제어 컴포넌트인 input을 제어 컴포넌트로 만들기 위한 훅입니다.

훅에 input value의 초기값을 인자로 제공하면 초기값을 설정할 수 있습니다.
value, onChange, reset함수를 통해 간편하게 입력값을 관리할 수 있습니다.
